# OpenSignals-Travel-Helper-CP

## Contenpack for the Opensignalsmod from MrTroble, Uhutown and Jeronimo.

**This content pack adds new Signals to Opensignals.**
The pack contains:

-  Travelhelper Signals
-  Zp9 Signals
-  Zp9 Control Column
-  Brake Test Signals
